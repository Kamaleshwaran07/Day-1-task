Objects

Objects are the most important data types in Javascript which is signficantly different from primitive data types which store single value for the keyword. Objects contain combination of multiple data types in the form of key:value pair. These keys can be variables or functions and are called properties and methods, respectively, in the context of an object.

Object Properties

Every object has some property associated with some value. These values can be accessed using these properties associated with them. var myCar = new object(); myCar.make = "TATA"; myCar.model = "Safari"; myCar.year = 1997; So now the values inside the object can be accessed by using keys i.e., mycar.year Output : 1997. These values can be accessed using brackets notation also. myCar[model] Output: Safari. The syntax for adding a property to an object is : Objectname.obejectProperty = propertyValue.
